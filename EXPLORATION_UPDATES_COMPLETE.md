## 🚀 How to Test

1. **Start the app**:
```bash
cd frontend && npm run dev
cd backend && npm run dev
```

2. **Navigate**: Home → Exploration → Click India → Click Kerala → "Explore Kerala"

3. **Test Features**:
   - Move character with arrow keys
   - Click landmarks (yellow markers) to see info
   - Click NPCs (teal circles) to read dialogues
   - Play all 4 games
   - Earn XP and level up
